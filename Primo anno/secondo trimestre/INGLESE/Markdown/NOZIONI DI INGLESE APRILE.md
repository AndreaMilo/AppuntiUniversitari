# LEZIONE 1: STRUMENTI DI MISURA E CONTENITORI

In questa lezione vediamo alcuni strumenti di laboratorio utilizzati per misurare o contenere liquidi e solidi.

### Cilindro Graduato (Measuring Cylinder)  
Il cilindro graduato è un contenitore di vetro cilindrico, vuoto all'interno, chiuso a un'estremità. Viene utilizzato per misurare i volumi di liquidi.  
**Esempio**:  
- "I used the measuring cylinder to measure 100 ml of water." (Ho usato il cilindro graduato per misurare 100 ml di acqua.)

### Provette (Test Tube)  
Una provetta è un breve tubo di vetro con un'estremità arrotondata, usato per contenere liquidi o solidi, in particolare in esperimenti di laboratorio.  
**Esempio**:  
- "The scientist placed the chemicals in the test tube." (Lo scienziato ha messo i chimici nella provetta.)

### Becher (Beaker)  
Un becher è un contenitore di vetro cilindrico, chiuso a un'estremità, usato per contenere liquidi o solidi in laboratorio.  
**Esempio**:  
- "The beaker contains the solution we need for the experiment." (Il becher contiene la soluzione di cui abbiamo bisogno per l'esperimento.)

# LEZIONE 2: AZIONI DEL MOUSE / INPUT DEVICES

Questa lezione riguarda le azioni di base che possiamo eseguire con il mouse durante l'uso di un computer.
### Click  
Fare clic su un elemento usando il tasto sinistro del mouse.  
**Esempio**:  
- "Click on the icon to open the application." (Clicca sull'icona per aprire l'applicazione.)
### Move  
Muovere il mouse per spostare il cursore sullo schermo.  
**Esempio**:  
- "Move the mouse to position the pointer over the text." (Muovi il mouse per posizionare il cursore sul testo.)
### Select  
Selezionare un elemento, come evidenziare una parola o cliccare su un'icona per aprire un programma.  
**Esempio**:  
- "Select the file you want to open." (Seleziona il file che vuoi aprire.)
### Control  
Usare il tasto "Ctrl" sulla tastiera per combinare azioni, come copiare e incollare.  
**Esempio**:  
- "Press Ctrl + C to copy the text." (Premi Ctrl + C per copiare il testo.)
### Drag  
Trascinare un oggetto tenendo premuto il tasto sinistro del mouse per spostarlo.  
**Esempio**:  
- "Drag the window to the top of the screen." (Trascina la finestra in alto sullo schermo.)
### Double-click  
Fare doppio clic su un oggetto per aprirlo o eseguire un'azione.  
**Esempio**:  
- "Double-click the file to open it." (Fai doppio clic sul file per aprirlo.)

### Keyboard (Tastiera)  
La tastiera è un dispositivo fondamentale per l'inserimento di testo e comandi nel computer.  
**Esempio**:  
- "I use the keyboard to type messages." (Uso la tastiera per scrivere messaggi.)

### Mouse (Mouse)  
Il mouse è utilizzato per muovere il cursore sullo schermo e interagire con gli oggetti.  
**Esempio**:  
- "I clicked the mouse to select the option." (Ho cliccato il mouse per selezionare l'opzione.)

### Light Pen (Penna Luminosa)  
La penna luminosa è un dispositivo che rileva la luce su uno schermo e viene usato per disegnare o selezionare oggetti sul monitor.  
**Esempio**:  
- "The artist used a light pen to draw on the screen." (L'artista ha usato una penna luminosa per disegnare sullo schermo.)

Vediamo come esprimere le funzioni e le capacità degli oggetti in inglese.

Per esprimere le capacità di un oggetto.

Indica il funzionamento di un oggetto.  
**Esempio**:

# LEZIONE 5: FORMA PASSIVA (Passive Voice)

### 🔹 Cos'è la forma passiva?
La **passiva** si usa quando vogliamo concentrarci su *chi subisce* l'azione invece di chi la compie.

- **Attivo**: Someone stole my bike.  
- **Passivo**: My bike was stolen.

### 🔹 Come si forma?
**Soggetto + to be (coniugato) + participio passato del verbo principale**

#### 📊 Esempi nei tempi verbali:
- **Present Simple**: The book **is read** by many students.  
- **Past Simple**: The letter **was sent** yesterday.  
- **Present Perfect**: The cake **has been eaten**.  
- **Future Simple**: The project **will be completed** soon.

# LEZIONE 7: PERIODI IPOTETICI (Conditionals)

### 🔹Zero Conditional
- **Uso**: Fatti sempre veri, leggi scientifiche.
- **Struttura**:  
  `If + Present Simple, Present Simple`
- **Esempio**:  
  `If you heat water to 100°C, it boils.`

### 🔹First Conditional
- **Uso**: Situazioni reali o possibili nel futuro.
- **Struttura**:  
  `If + Present Simple, will + verbo`
- **Esempio**:  
  `If it rains, we will stay inside.`

### 🔹Second Conditional
- **Uso**: Situazioni ipotetiche o improbabili nel presente/futuro.
- **Struttura**:  
  `If + Past Simple, would + verbo`
- **Esempio**:  
  `If I had a car, I would drive to work.`

### 🔹Third Conditional
- **Uso**: Situazioni ipotetiche nel passato, spesso con rimpianti.
- **Struttura**:  
  `If + Past Perfect, would have + participio passato`
- **Esempio**:  
  `If I had studied, I would have passed the exam.`

# LEZIONE 8: COMPUTER DEVICES AND COMPONENTS

## Mouse

**IT:** Il **mouse** è un dispositivo di input che consente di selezionare, cliccare e trascinare elementi sullo schermo del computer.

**EN:**  
A **mouse** is an input device used to control the pointer on a computer screen. It allows users to click, select, and drag objects.

It typically has:
- Two buttons (left and right)
- A scroll wheel between them
- An ergonomic shape
- A USB cable or wireless connectivity (Bluetooth or RF)

It is placed next to the keyboard on a flat surface, often on a mouse pad.

**Esempio:**  
- "Move the mouse to select the icon." (Muovi il mouse per selezionare l'icona.)

## Main Computer Components

Questi sono i componenti principali che costituiscono un computer da scrivania:

- ### Monitor
	The monitor is an output device that displays everything the computer processes. Most monitors connect via VGA, DVI, HDMI, or DisplayPort.
	![[Pasted image 20250424181641.png]]
	

	**Esempio:**  
- "The monitor shows the desktop." (Il monitor mostra il desktop.)

- ### System Unit (Case)
	The main body of the computer contains:
	- **Motherboard**: the central board that connects all components.
	- **CPU (Processor)**: performs all instructions.
	- **RAM (Memory)**: temporarily stores active data.
	- **Storage**: hard disk (HDD) or solid state drive (SSD).
	- **GPU**: processes visual output.
	- **Power supply (PSU)**: distributes power.

- ### Keyboard
	The keyboard is an input device used to type commands, numbers, and letters.

	Common keys include:
	- Function keys (F1–F12)
	- Control (Ctrl), Alt, and Shift
	- Arrow keys, Spacebar, Backspace

	**Esempio:**  
- "Use the keyboard to type a message." (Usa la tastiera per scrivere un messaggio.)

## Connecting Components

Come collegare correttamente i dispositivi è essenziale per il funzionamento del computer.
### Monitor
Connect the video cable (VGA, DVI, HDMI) to the monitor and the back of the PC. Plug in the power cable. 
*Collegare il cavo video (VGA, DVI, HDMI) al monitor e al retro del PC. Collegare il cavo di alimentazione.*
### Mouse and Keyboard
Connect them via USB ports or PS/2 ports (round connectors: green for mouse, purple for keyboard).
*Collegateli tramite porte USB o porte PS/2 (connettori rotondi: verde per il mouse, viola per la tastiera).*
### Internet Connection
- Via **Ethernet cable** for a wired connection. $\rightarrow$  Tramite **cavo Ethernet** per una connessione cablata.
- Via **Wi-Fi** if the computer has a wireless adapter. $\rightarrow$ Tramite **Wi-Fi** se il computer è dotato di un adattatore wireless.

## Accesso al Sistema

L'accesso al computer avviene tramite una procedura di login:
1. Turn on the computer.
2. The cursor appears on the screen.
3. Type `LOGIN` and press `RETURN`.
4. Enter the user ID and then the password.
5. Once authenticated, choose your preferred language.
6. The system is now ready for use.

***Traduzione***:
1. Accendere il computer.
2. Il cursore appare sullo schermo.
3. Digitare `LOGIN` e premere `INVIO`.
4. Inserire l'ID utente e poi la password.
5. Una volta autenticato, scegliere la lingua preferita.
6. Il sistema è ora pronto per l'uso.

**Esempio:**  
- "Log in with your ID and password to start working." (Accedi con ID e password per iniziare a lavorare.)
