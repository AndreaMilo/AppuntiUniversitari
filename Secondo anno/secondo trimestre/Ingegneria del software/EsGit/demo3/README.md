\# Tim Berners-Lee (Vincitore 2016) 



\## Profilo: 
Informatico e fisico britannico, noto soprattutto come l'inventore del World Wide Web.



\## Contributo Chiave: 
Nel 1989, mentre lavorava al CERN di Ginevra, ha proposto la creazione di un sistema di gestione delle informazioni basato sull'ipertesto, che è diventato la base fondamentale del web moderno.



\## Perché ha vinto: 
L'ACM (Association for Computing Machinery) gli ha conferito il premio per aver inventato il World Wide Web, il primo browser web e i protocolli e algoritmi fondamentali (come HTTP e HTML) che permettono al web di scalare.



\## Impatto:
Il suo lavoro ha rivoluzionato il modo in cui l'umanità condivide informazioni, rendendo l'accesso ai dati universale e interconnesso.
