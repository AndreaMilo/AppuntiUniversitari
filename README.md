# AppuntiUniversitari
Appunti presi durante la mia permanenza a Informatica L31 Uniba Aldo Moro

Benvenuto/a nel mio repository di appunti! 
Questo file è stato generato come test per verificare il corretto funzionamento della sincronizzazione tra **Obsidian** e **GitHub** su ambienti Windows e Linux. Se riesco a leggere questo testo su tutti i miei dispositivi, significa che il setup ha funzionato! 🎉

## 📂 Struttura del Vault
Qui raccoglierò tutti i miei appunti, divisi per materie e argomenti. La struttura generale è (o sarà) la seguente:

* **Matematica e Scienze di Base:** Analisi, Algebra, ecc.
* **Informatica:** Programmazione, Architettura degli Elaboratori, Database.
* **Progetti e Laboratori:** Codice, schemi e relazioni.
* **Esami Passati:** Domande frequenti e simulazioni.

## ⚙️ Dettagli Tecnici
* **Editor:** [Obsidian](https://obsidian.md/)
* **Sync:** Gestito tramite il plugin *Obsidian Git*
* **Status:** Lavori in corso 🚧
